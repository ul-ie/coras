Flexbox vs Grid

Our layout system uses both flexbox and grid, depending on the use case.

Choose flexbox if:

  • You have just a few items to arrange.
  • Centering items when they wrap onto a new row is important.

Choose grid if:

  • You have multiple items on a page.
  • Consistent gutter/gap spacing is needed.
  • Or, if the layout is non-standard (e.g. a 'Bento Box' layout where some items need to span multiple rows and/or columns).
