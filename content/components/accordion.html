Description

An accordion is an expandable and collapsable container component that reveals its content when the user interacts with it. Accordions help organise information, and can help users consume the information they need more efficiently.

Syntax

An accordion should be used with the embed tag, as it is a container for other content. To place content inside an accordion, use the block placeholder named acc_html as follows:

{{ html("
{% embed 'THEME_NAME:accordion' with {
  property: 'value' } %}

  {% block acc_html %}

    

  {% endblock %}

{% endembed %}
")}}

Properties

The accordion component has six properties in full, but only two are required: acc_title and acc_html.

Variants

The accordion component has several properties that allow for customisation, but this customisation should only be availed of in exceptional circumstances, e.g. if the accordion is placed on a coloured background.

Title Length

The accordion title is not a replacement for a page heading and should not be used as such. Titles should be kept short and concise.

Title Text Size

The font size of the accordion titles aligns with that of a h3 element, but it can be adjusted if required using the acc_title_size property. The value is medium by default, and can be set to large or small. It might be changed to small for instance if you had an FAQ section and many of the questions needed to flow onto two or more lines.

Title Colour

The acc_title_colour property can be used if it is necessary to toggle the colour of both the title text and chevron icon from green to white. This can be useful if the accordion needs to be placed on a coloured background.
