Description

The icon component can be used standalone or nested inside another component, such as a button or, in the below example, a well.

{% embed "@components/well/well.twig" with { well_modifiers: 'inline-block leading-none' } %} {% block well_html %} {% include '@components/icon/icon.twig' with { icon_name: 'user', icon_size: 'w-10', icon_fill: 'fill-green dark:fill-modern' } %} {% endblock %} {% endembed %}

Syntax

Use the include tag to inject an icon. The icon must exist in the UL custom icon kit for it to be added to the icon bank (see Iconography).

{{ html("
{% include 'THEME_NAME:icon' with {
  property: 'value'
  ...
} %}
")}}

Properties

An icon has one mandatory and four optional properties.

{% include '@templates/component-props.html' %}

Variants

Icons can be customised by colour, size and type.

Colour

All icon colours are defined by the icon_fill property, which expects a class or set of classes. Styling the stroke will have no effect; even very thin icons do not have a stroke.

Note: Twotone icons are styled by a single colour; the secondary path (the one with the lighter colour) simply has a lower opacity.

Examples

(i) Setting the icon_fill of a graduation cap to UL Green and, if dark mode is supported, to Modern Green:

{% include '@components/icon/icon.twig' with { icon_name: 'graduation-cap', icon_fill: 'fill-green dark:fill-modern', icon_size: 'w-20' } %} 
{{ html("
{% include 'ul:icon' with {
  icon_name: 'graduation-cap',
  icon_fill: 'fill-green dark:fill-modern'
} %}
")}}

(ii) Setting the icon_fill to Shannon, and using a lighter shade of the same colour on hover.

{% include '@components/icon/icon.twig' with { icon_name: 'earth-europe', icon_fill: 'fill-shannon hover:fill-shannon-l1', icon_size: 'w-16' } %} 
{{ html("
{% include 'ul:icon' with {
  icon_name: 'earth-europe',

} %}
")}}

Size

The icon_size will default to w-4 (1 rem) if left unset. Below are some examples of increasing this size to w-6, w-8 and w-10 respectively. For brevity, only the code for the last example is shown.

{% include '@components/icon/icon.twig' with { icon_name: 'user', icon_fill: 'fill-golden' } %} {% include '@components/icon/icon.twig' with { icon_name: 'user', icon_fill: 'fill-golden', icon_size: 'w-6' } %} {% include '@components/icon/icon.twig' with { icon_name: 'user', icon_fill: 'fill-golden', icon_size: 'w-8' } %} {% include '@components/icon/icon.twig' with { icon_name: 'user', icon_fill: 'fill-golden', icon_size: 'w-10' } %} 
{{ html("

  } %}
")}}

Icons can also be sized by height, e.g. using a h-10 class (which features in the brand examples below).
