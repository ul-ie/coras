Description

The icon component can be used standalone or nested inside another component, such as a button or, in the below example, a well. It uses a custom icon() function under the hood that mirrors the functionality of the Drupal Icon API.

{% embed "@components/well/well.twig" with { modifiers: 'inline-block leading-none' } %} {% block html %} {% include '@components/icon/icon.twig' with { name: 'user', size: 'w-10', fill: 'fill-green dark:fill-modern' } only %} {% endblock %} {% endembed %}

To see all available icons, visit the Iconography page.

Syntax

Use the include tag to inject an icon, and the only keyword if it needs to be protected from outside scope.

{{ html("
{% include 'ul:icon' with {
  property: 'value'
  ...
} only %}
")}}

Properties

An icon has one mandatory and seven optional properties.

Variants

Icons can be customised by colour, size and type.

Colour

All icon colours are defined by the fill property, which expects a class or set of classes. Styling the stroke will have no effect; even very thin icons do not have a stroke.

Note: Twotone icons are styled by a single colour; the secondary path (the one with the lighter colour) simply has a lower opacity.

Examples

(i) Setting the fill of a graduation cap to UL Green and, if dark mode is supported, to Modern Green:

{% include '@components/icon/icon.twig' with { name: 'graduation-cap', fill: 'fill-green dark:fill-modern', size: 'w-20' } only %} 
{{ html("
{% include 'ul:icon' with {
  name: 'graduation-cap',
  fill: 'fill-green dark:fill-modern'
} %}
")}}

(ii) Setting the fill to Shannon, and using a lighter shade of the same colour on hover.

{% include '@components/icon/icon.twig' with { name: 'earth-europe', fill: 'fill-shannon hover:fill-shannon-l1', size: 'w-16' } %} 
{{ html("
{% include 'ul:icon' with {
  name: 'earth-europe',
  fill: 'fill-shannon hover:fill-shannon-l1'
} %}
")}}

Size

The size will default to w-5 (1.25 rem) if left unset. Below are some examples of increasing this size to w-7, w-9 and w-11 respectively. For brevity, only the code for the last example is shown.

{% include '@components/icon/icon.twig' with { name: 'user', fill: 'fill-golden' } %} {% include '@components/icon/icon.twig' with { name: 'user', fill: 'fill-golden', size: 'w-7' } %} {% include '@components/icon/icon.twig' with { name: 'user', fill: 'fill-golden', size: 'w-9' } %} {% include '@components/icon/icon.twig' with { name: 'user', fill: 'fill-golden', size: 'w-11' } %} 
{{ html("
{% include 'ul:icon' with {
  name: 'user',
  fill: 'fill-golden',
  size: 'w-10'
} %}
")}}

Icons can also be sized by height, e.g. using a h-10 class (which features in the social examples below).
