Description

A well is a very simple component designed to nest other content or components.

{% embed '@components/well/well.twig' %} {% block html %}
div content inside a well.
{% endblock %} {% endembed %}

Syntax

A well is to be used with the embed tag, as it is a container for other content. To place content inside the well, use the block placeholder named html as follows:

{{ html("
{% embed 'ul:well' with {
  property: 'value' } %}

  {% block html %}

    

  {% endblock %}

{% endembed %}
")}}

Properties

In addition to the html block a well has two properties, both optional.

{% include "@layouts/component-props.html" %}

Variants

The padding can be customized per instance using the optional padding variable. In the following examples the padding is indicated with a dashed outline.

Custom Padding

To decrease or increase the padding, use a different value for the utility padding class, e.g. p-2 to make it smaller:

{{ html("
{% embed 'ul:well' with { padding: 'p-2' } %}

  {% block html %}

    

  {% endblock %}

{% endembed %}
")}}

This would result in the following output:

{% embed '@components/well/well.twig' with { padding: 'p-2' } %} {% block html %}
A well with less padding.
{% endblock %} {% endembed %}

Conditional Padding

You could also have the padding adapt to different screen sizes. For example, in the following code the padding will change depending on the size of the user's screen:

{{ html("
{% embed 'ul:well' with { 
  padding: 'p-3 md:p-5 lg:p-8' 
} %}

  {% block html %}

    

  {% endblock %}

{% endembed %}
")}}
{% embed '@components/well/well.twig' with { padding: 'p-3 md:p-5 lg:p-8' } %} {% block html %}
Responsive padding.
{% endblock %} {% endembed %}

Modifier Classes

Using the modifiers property, you can further adjust the appearance of the well. For example, if you wanted it to appear as an inline block rather than the default block, you could use the following code:

{{ html("
{% embed 'ul:well' with { 
  modifiers: 'inline-block' 
} %}

  {% block html %}

    

  {% endblock %}

{% endembed %}
")}}
{% embed '@components/well/well.twig' with { modifiers: 'inline-block' } %} {% block html %}
A well styled as an inline block.
{% endblock %} {% endembed %}
