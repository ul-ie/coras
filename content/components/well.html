Description

A well is a very simple component designed to nest other content or components. It consists of a border with a slightly rounded radius, some padding and a subtle background colour to distinguish it from surrounding content.

{% embed '@components/well/well.twig' %} {% block well_html %}
div content inside a well.
{% endblock %} {% endembed %}

Syntax

A well is to be used with the embed tag, as it is a container for other content. To place content inside the well, use the block placeholder named well_html as follows:

{{ html("
  {% embed '@components/well/well.twig' %}

    {% block well_html %}

      

    {% endblock %}

  {% endembed %}
  ")}}

Variants

The padding can be customized per instance using the optional padding parameter. In the following examples the padding is indicated with a dashed outline.

Custom Padding

To decrease or increase the padding, use a different value for the utility padding class, e.g. p-2 to make it smaller:

{{ html("
  {% embed '@components/well/well.twig' with { well_padding: 'p-2' } %}

    {% block well_html %}

      

    {% endblock %}

  {% endembed %}
  ")}}

This would result in the following output:

{% embed '@components/well/well.twig' with { well_padding: 'p-2' } %} {% block well_html %}
A well with less padding.
{% endblock %} {% endembed %}

Conditional Padding

You could also have the padding adapt to different screen sizes. For example, in the following code the padding will change depending on the size of the user's screen:

{{ html("
  {% embed '@components/well/well.twig' with { 
    well_padding: 'p-3 md:p-5 lg:p-8' 
  } %}

    {% block well_html %}

      

    {% endblock %}

  {% endembed %}
  ")}}
{% embed '@components/well/well.twig' with { well_padding: 'p-3 md:p-5 lg:p-8' } %} {% block well_html %}
Responsive padding.
{% endblock %} {% endembed %}

Modifier Classes

Using the well_modifiers property, you can further adjust the appearance of the well. For example, if you wanted it to appear as an inline block rather than the default block, you could use the following code:

{{ html("
  {% embed '@components/well/well.twig' with { 
    well_modifiers: 'inline-block' 
  } %}

    {% block well_html %}

      

    {% endblock %}

  {% endembed %}
  ")}}
{% embed '@components/well/well.twig' with { well_modifiers: 'inline-block' } %} {% block well_html %}
A well styled as an inline block.
{% endblock %} {% endembed %}
