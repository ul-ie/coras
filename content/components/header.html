Scroll Behaviour

This example is populated with sample content so that you can scroll up and down the page. When the user scrolls down, the header disappears. When the user scrolls back up, it appears again. The distance from the top of the page is also monitored to alter the size of the header.

Space Above

The header is not fixed but has a sticky position that ensures it does not obscure any elements above it (e.g. contextual menus when logged in).
