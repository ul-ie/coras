{{ title }}

Description

The profile card component is primarily for academic profiles that link to the Pure research portal.

{% include '@components/profile/profile.twig' with { title: 'Title(s)', name: 'Name', profile_url: '#', img_src: '/img/profiles/profile-example-good.svg' } %}

Syntax

The profile card component has four parameters:

  • name: The full name of the person.
  • title: Their title(s).
  • img_src: The URL of the image to use (defaults to an SVG icon if not provided).
  • profile_url: The external URL to their Pure profile.

Use the include tag to render the profile with its parameters as follows:

{{ html('
        {% include "@components/profile/profile.twig" with {
            name: "Full Name",
            title: "Dr., Associate Professor",
            profile_url: "/url/to/pure/profile",
            img_src: "/url/to/profile/img"
        } %}
    ') }}
